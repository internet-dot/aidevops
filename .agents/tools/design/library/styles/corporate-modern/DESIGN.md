# Design System: Corporate Modern

Contemporary enterprise aesthetic — confident, clean, forward-looking. Targets modern SaaS platforms, fintech, and tech-forward professional services. Teal primary (`#0891b2`), charcoal text (`#1a1a2e`), white surfaces, 8px spacing grid, 150ms ease-out transitions.

## Quick Reference

**Colours:** Primary `#0891b2` · Text `#1a1a2e` · Body `#334155` · Background `#FFFFFF` · Surface `#F8FAFC` · Border `#E2E8F0`

**Shapes:** Buttons/inputs `border-radius: 8px` · Cards `border-radius: 12px` · Pills `border-radius: 9999px`

**Animation:** `transition: all 150ms ease-out`

**Typography:** Inter / system-ui · Headings weight 700 · Body weight 400 · Max content width 720px
